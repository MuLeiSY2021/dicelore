## A. 业务需求汇总（subagent 收集，已去重聚类）

### 主页（新玩家）
- P0 醒目「开新局」入口；空状态引导；继续上次大卡片(团本·幕·角色HP/钱/进度·时间)；「继续跑团」一键续；无活动会话时「跑团」导航置灰。
- P1 最近 Session 列表(状态+时间，可点入)；四快速入口卡片；顶 bar 运行态指示(模型/MCP/notify)；明暗+语言切换。
- P2 时段欢迎语+剧情一句话；会话管理搜索/续/删；新手说明入口。

### 跑团页（新玩家+资深玩家）
- P0 中央叙事散文；自由文本输入开回合；明骰塌成单「丢骰子」按钮；掷骰前区间预览；掷骰后结果+成败高亮；choice 选项点选(需补 postChoice 闭环)；呈现台核心状态(HP/钱/声望)；空状态友好引导。
- P1 倒计时钟圆盘(2/8 分段+名+触发说明)；库存面板；揭示卡自动钉(标 GM·回合号)；生成中/收尾态反馈(turn_started/ended)；左活动轨四源(设定/规则/日志/会话)浏览；树形搜索+pin 到呈现台；叙事内嵌机械回显；错误提示(error)。
- P2 叙事/打字分隔条拖拽；面板拖拽/缩放/最小化/隐藏(面板三态)；终局画面(game_end)；活动轨拖出生成面板。

### 团本制作页（团本作者）
- P0 选团本列表(GET /catalog ✓)；中央按类型即写即读渲染；NPC 卡片人设散文+sheet卡；缺卡 warn；构建助手对话(POST /lore-sessions ✓)+显示调了哪些工具；整包校验报告(error/warn 计数+定位文件，validatePack ✓)；校验整包按钮。
- P1 新建/选择团本不丢草稿；左导航六类型+计数；构建进度阶段点；NPC 折叠/展开；导出团本包(commit ✓)；发布 tag(✓)；草稿版本号。
- P2 import 原著切块检索库(后端未起，**本轮不做**)；inline CRUD 写(构建期 MCP 未暴露，**本轮走助手对话**)。

### 配置页 7 子页（配置者）—— 用户特别强调「全是假的、要做真」
- 通用：语言切换(简中/English)真生效+持久化；启动行为。
- 服务与网络：展示后端真实端口/notify URL+连通状态(diagnostics)。
- MCP 服务器：dicelore 真实工具数/notify 探测(锁定必需)；自定义 MCP 增删改+开关+权限闸+持久化；**连接测试真握手**；out-of-canon 警示。
- 模型连接：**GM 模型下拉真实列表**(opus-4-8/sonnet/haiku)；baseURL/key 输入(掩码+持久化)；**连接测试真测通断**；agent 可选。
- 主题外观：主题/明暗(+跟随系统)/强调色/字体，持久化。
- 数据与存储：真实会话目录/FTS 模式+统计。
- 关于：真实版本+protocol 版本。
- 贯穿：**所有配置持久化**(改了下次还在)。

## B. 范围裁定（诚实边界）

**做真（后端今天支持或加薄端点）**：
- 前端：Logo(d20宝石SVG)、i18n(zh/en)、主题持久化、Settings(localStorage)、全配置子页真实交互、跑团页对齐草图、团本页接真 catalog。
- 后端薄端点：`GET /diagnostics/health`(服务器真值)、`POST /diagnostics/model-test`、`POST /diagnostics/mcp-test`、`GET /sessions/:id/browse?source=`(loreSearch/ruleSearch/logSince/stateList)、`GET /catalog/:id/files?ref=`(checkout)、`POST /catalog/validate`(validatePack)、`GET/PUT /settings`(env 真值读 + 覆盖落盘)。

**本轮不做（后端未起/明确推迟，wiki 主题H/P5）**：import 原著切块、构建期 inline CRUD MCP、per-instance roll gate、BG3 动效精修、Tauri 壳、Web 鉴权、game_end 终局画面(后端无映射，仅前端兜底提示)。
